## 요구 사항

### 1. 다이어리

- 일정 달력에 다이어리 추가
- 다이어리엔 사진, 영상 등 다양한 미디어 파일도 업로드/다운로드 가능해야함
- 일정달력 또는 리스트식으로 다이어리
- 다이어리에 적을 내용은 옵션으로 선택가능하게 해볼까
  - 제목, 내용, 날짜는 필수
  - 장소(map 이랑 연동), 사진, 영상, gif



### (1) 메인 페이지

- navbar - 프로젝트이름, 아이콘들(마이페이지, 일정, 친구, 채팅, 등등)
- 달력있고 날짜 클릭하면 옆에 TodoLIst나 다이어리 목록 띄우기 - 다이어리 클릭하면 해당 다이어리로 이동
- 연휴도 추가해야하고.. 
- 날짜는 1900년-2100년?
