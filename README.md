## 요구 사항

### 1. 다이어리

- 일정 달력에 다이어리 추가
- 다이어리엔 사진, 영상 등 다양한 미디어 파일도 업로드/다운로드 가능해야함
- 일정달력 또는 리스트식으로 다이어리
- 다이어리에 적을 내용은 옵션으로 선택가능하게 해볼까
  - 제목, 내용, 날짜는 필수
  - 장소(map 이랑 연동), 사진, 영상, gif
